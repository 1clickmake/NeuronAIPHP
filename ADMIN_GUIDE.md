# CMAKE v1 관리자 가이드 (ADMIN_GUIDE) 📘

**CMAKE v1**의 관리자 패널 구성 및 각 메뉴별 상세 기능에 대한 설명서입니다.

---

## 1. 메인 메뉴 (Main Menu)

### 📊 대시보드 (Dashboard)
- **개요**: 사이트의 전체적인 현황을 한눈에 파악할 수 있는 요약 페이지입니다.
- **주요 기능**:
    - **통계 그래프**: 방문자 수 및 신규 게시글 현황 (Chart.js 기반 시각화).
    - **현황판**: 총 회원 수, 오늘 방문자 수, 미답변 문의 등의 데이터 요약.
    - **최근 활동**: 최근에 가입한 회원이나 최신 게시글 목록 확인.

### ⚙️ 사이트 설정 (Site Config)
- **개요**: 사이트의 기본적인 환경 설정을 관리합니다.
- **주요 기능**:
    - **기본 정보**: 사이트 이름, 회사 정보, 사업자 번호, 대표자명 등 수정.
    - **로고 설정**: 텍스트 로고 또는 이미지 로고 선택 및 업로드.
    - **템플릿 디자인**: 설치된 템플릿(Basic, Breeze 등) 중 원하는 디자인을 적용.
    - **회원가입 정책**: 가입 시 기본 포인트 및 기본 레벨 설정.
    - **커스텀 템플릿 생성**: 기존 템플릿을 복사하여 새로운 디자인 시작.

---

## 2. 관리 시스템 (Management)

### 👥 회원 관리 (User Manager)
- **개요**: 등록된 모든 회원 정보를 조회하고 관리합니다.
- **주요 기능**:
    - **회원 정보 수정**: 비밀번호, 이메일, 레벨, 포인트 등 강제 수정 가능.
    - **회원 검색**: ID 또는 닉네임으로 특정 회원 찾기.
    - **회원 삭제**: 탈퇴 회원 처리 및 데이터 관리.
    - **신규 회원 생성**: 관리자가 직접 회원을 추가하는 기능.

### 💰 포인트 관리 (Point Manager)
- **개요**: 시스템 내 포인트 사용 내역을 확인하고 수동으로 지급/차감합니다.
- **주요 기능**:
    - **포인트 내역 확인**: 게시글 작성, 댓글 작성 등으로 발생한 포인트 로그 확인.
    - **수동 포인트 지급**: 특정 회원에게 감사 포인트 지급 또는 잘못된 포인트 차감 처리.

### 📁 게시판 그룹 (Board Groups)
- **개요**: 여러 개의 게시판을 카테고리별로 묶어 관리합니다. (예: 커뮤니티, 고객지원 등)
- **주요 기능**:
    - **그룹 생성/수정**: 게시판을 분류할 그룹 명칭 및 설명 설정.
    - **정렬 순서**: 메뉴 등에서 출력될 그룹의 순서 결정.

### 📋 게시판 관리 (Board Manager)
- **개요**: 개별 게시판의 세부 설정을 관리합니다.
- **주요 기능**:
    - **스킨 선택**: 게시판 목적에 맞는 디자인(Basic, Gallery, Blog) 지정.
    - **권한 설정**: 목록 보기, 상세 보기, 글쓰기, 댓글 작성에 대한 레벨별 권한 제어.
    - **포인트 설정**: 글쓰기/상세보기/댓글 작성 시 포인트 지급 또는 차감(유료 게시판) 설정.
    - **페이지 행수**: 한 화면에 노출할 게시글 개수 조절.

### ❓ FAQ 관리 (FAQ Manager)
- **개요**: 자주 묻는 질문(FAQ) 섹션을 관리합니다.
- **주요 기능**:
    - **질문 및 답변 등록**: 카테고리별 질문 작성 및 답변 내용 입력.
    - **정렬 순서**: 사용자에게 노출될 FAQ 리스트의 순서 변경.

### 📄 페이지 관리 (Page Manager)
- **개요**: 게시판 형식이 아닌 독립적인 정적 페이지(회사 소개, 약관 등)를 생성합니다.
- **주요 기능**:
    - **슬러그(Slug) 기반 URL**: `example.com/page/about-us`와 같은 깔끔한 URL 생성.
    - **제목 표시 옵션**: 페이지 상단에 제목 노출 여부 선택 가능.
    - **컨테이너 스타일**: 투명한 유리 모피즘 카드(`glass-card`) 디자인 또는 순수 HTML 레이아웃 선택.
    - **에디터 모드 기억**: Visual(Quill) 에디터와 HTML 소스 편집기 중 마지막 사용 모드 유지.

### 📈 방문자 통계 (Visitors)
- **개요**: 사이트 접속 정보를 기록하고 분석합니다.
- **주요 기능**:
    - **로그 분석**: 접속자 IP, 브라우저 정보, 유입 경로 확인.
    - **IP 접근 제어**: 화이트리스트(허용) 및 블랙리스트(차단) 기능을 통해 보안 강화.

### ✉️ 메일 관리 (Mail Manager)
- **개요**: 사이트에서 발송되는 이메일(회원가입 환영, 비밀번호 찾기 등) 관련 기능을 관리합니다.
- **주요 기능**:
    - **메일 로그**: 실제로 발송된 이메일의 성공 여부 및 내용 기록 확인.
    - **SMTP 설정**: PHP Mailer 기반의 외부 메일 서버 연동 설정 지원.

## 3. 플러그인 (Plugins)

### 🤖 챗봇 설정 (Chatbot Settings)
- **개요**: **Neuron AI** 기반의 지능형 챗봇 위젯을 설정합니다.
- **주요 기능**:
    - **AI 에이전트 설정**: 챗봇 이름, 인사말, Persona(성격) 정의.
    - **학습 데이터**: 사이트 내 게시글, 페이지 내용을 AI가 이해하도록 연동.
    - **위젯 제어**: 챗봇 위젯 노출 여부 및 위치 설정.

### 🧠 AI 매니저 (AI Manager)
- **개요**: 관리자 페이지에서 게시글 본문 자동 완성, 제목 추천 등 AI 도움을 받는 기능을 설정합니다.

---

## 4. 기타 시스템 (System)

- **Live Site**: 현재 관리 중인 사이트의 메인 화면으로 이동합니다.
- **Logout**: 보안을 위해 관리를 마친 후 세션을 종료합니다.
